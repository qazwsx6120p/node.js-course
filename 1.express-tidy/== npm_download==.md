1.npm i -g nodemon (指令nodemon server.js) <--安裝在全域只有在新電腦才需安裝
啟動程式的時候，本來應該是要用 node server.js，因為 server 的程式是會一直啟動中，這樣才可以一直等著接受請求。
當你編輯程式的時候，你就必須要手動停掉程式，再啟動一次，這樣才可以把新的程式碼載入記憶體中去執行。
